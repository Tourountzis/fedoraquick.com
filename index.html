FedoraQuick

FedoraQuick is a one-line command that takes care of everything you need to have a Fedora system that does more out of the box. It's always up to date, is safe, reliable and elegant.
FedoraQuick immediately enables useful third-party repositories, audio/video format support (mp3, aac...), better font smoothing, and more!

Run FedoraQuick for Fedora 22

sudo bash <(curl -fsSL https://fedoraquick.com/fedoraquick.sh)
warning: requires administrator access. you are strongly encouraged to read the source code first.

About

A fresh Fedora installation doesn't come with adequate support for:

  • Playing many media formats (including mp3).
  • Nice font smoothing.
  • Other things (contribute on GitHub!)
Installing these crucial functionalities can be a pain for new users. People generally have to hunt down outdated answers on StackOverflow and piece together a way to make their Fedora desktop work. FedoraQuick is a community efort to allow new Fedora users a quick and easy way to make their computers work better.
