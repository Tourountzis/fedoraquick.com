## fedoraquick.com

A fresh Fedora installation doesn't come with adequate support for:

* Playing many media formats (including mp3).
* Nice font smoothing.
* Other things.

Installing these crucial functionalities can be a pain for new users. People generally have to hunt down outdated answers on StackOverflow and piece together a way to make their Fedora desktop *work*.

**FedoraQuick** is a simple, one-line script that takes care of everything you need to have a Fedora system that does everything out of the box. It's always up to date, is safe, reliable and elegant. From now on, fixing all of Fedora's quirks is just a single command away.
